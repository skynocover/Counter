# GolangSyncCounter

[![](https://godoc.org/github.com/nathany/looper?status.svg)](https://pkg.go.dev/github.com/skynocover/Counter?tab=doc)

## How to use

### Before Use

```go
var Receive = counter.Counter{}
```

### Add one time
```go
Receive.Add()
```

### Reset Counter
```go
Receive.Reset()
```

### Counter now
```go
Receive.Time()
```
