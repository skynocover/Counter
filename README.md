# Counter
GolangSyncCounter

## How to use

### Before Use

```go
var Receive = counter.Counter{}
```

### Add one time
```go
Receive.Add()
```

### Reset Counter
```go
Receive.Reset()
```

### Counter now
```go
Receive.Time()
```
